## 2. B1934 - 최소공배수 + 유클리드 호제법

```python
T = int(input())

for _ in range(T):
    a, b = map(int, input().split())
    M = a * b 
    m = 0
    while a != 0 and b != 0:
        if a > b:
            a = a % b
            m = b
        else:
            b = b % a
            m = a
    print(int(M / m))
```
